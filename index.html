What is the World Wide Web?

The World Wide Web (WWW) is a vast network of digital information, where documents and media are interconnected through links and unique addresses. It was invented by Tim Berners-Lee in 1989 and operates through protocols like HTTP (Hypertext Transfer Protocol). The web allows users to view web pages, navigate between them, and interact with various types of online content.

Term Definiton
Web Browser A program that lets users access websites on the internet. It retrieves files from a web server and displays them on a user's device
Web Server A computer program or hardware that delivers content to users over the internet
