HTML Timeline

HTML started out as a simple way to organize text and links on webpages back in the 90s. As the web grew, so did HTML, adding features such as tables and forms in later versions. The biggest update came with HTML5 in 2014, which brought in features like support for video, better organization of content, and tools for creating more interactive and responsive websites. Now, HTML5 is the standard for building modern websites.

HTML 1.0 (1991)

<p>First paragraph</p>
<a href="next.html">Next page</a>
  • 18 basic tags
  • Hyperlinks between documents

HTML 2.0 (1995)

<form action="submit">
  <input type="text" name="email">
</form>
  • Forms for user input
  • Basic table support

HTML 3.0 (1997)

<table border="1">
  <tr><td>Cell 1</td></tr>
</table>

<font size="+1">Big text</font>
  • Advanced tables
  • Font styling (deprecated now) in favor of CSS Style Sheets
  • Early CSS support

HTML 4.0 (1997)

<link rel="stylesheet" href="style.css">

<div id="header">
  <h1>Title</h1>
</div>
  • Full CSS integration
  • Framesets (now deprecated)
  • Strict separation of structure/presentation

HTML5 (2014)

<video controls>
<source src="movie.mp4"type="video/mp4">
</video>

<article>
  <header>...</header>
</article>
  • Semantic elements (header, nav, section)
  • Native multimedia
  • Canvas/WebGL graphics
